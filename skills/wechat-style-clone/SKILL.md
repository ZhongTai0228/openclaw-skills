# WeChat Style Clone

克隆任意公众号文章的排版风格，用于排版新文章。

## 工作流程

### Phase 1: 提取风格

**输入**: 公众号文章 URL（`mp.weixin.qq.com/s/...`）

1. 用 `browser` 打开文章页面（profile=openclaw）
2. 等待 5 秒加载
3. 通过 JS 提取 `#js_content` 的 innerHTML
4. 分析 HTML，提取以下风格要素：

**必须提取的要素**:

| 类别 | 要素 |
|---|---|
| **全局** | 最大宽度、背景色、字体族、全局行高、字间距 |
| **颜色** | 主色调、辅助色、文字色系（标题/正文/辅助/极浅） |
| **正文** | font-size、line-height、color、text-align、margin-bottom |
| **标题** | 各级标题的样式（字号、字重、颜色、间距、装饰元素） |
| **高亮** | 所有高亮方式（渐变、背景色、下划线、加粗变色等） |
| **图片** | 圆角、边框、阴影、卡片包裹方式 |
| **特殊组件** | 引用块、列表、卡片、分隔线、互动区等 |
| **间距** | 段落间距、章节间距、图片间距规律 |
| **装饰** | 渐变条、圆点、分隔线等装饰元素 |

5. 将风格保存为模板文件：`references/style-profiles/<name>.md`

### Phase 2: 应用风格

**输入**: Markdown 内容（来自飞书文档或手写）+ 风格模板名

1. 读取风格模板
2. 将 Markdown 内容映射到风格模板的组件：
   - `#` / `##` → 对应标题样式
   - 正文段落 → 正文样式
   - `**加粗**` → 高亮样式（根据语境选择最合适的高亮方式）
   - `![图片]()` → 图片卡片样式
   - `> 引用` → 引用块样式
   - 列表 → 列表样式
3. 生成带 inline CSS 的 HTML 文件
4. 后续粘贴到公众号编辑器（可复用 agentverse-wechat 的 Phase 5-8 流程）

### Phase 3: 粘贴到公众号（可选）

如果用户需要直接发布，复用 `agentverse-wechat` skill 的粘贴流程：
- 本地起 HTTP 服务
- 浏览器打开 → 选中 → 复制
- 打开公众号编辑器 → 粘贴
- 设封面、摘要、预览

## 风格模板格式

每个风格模板保存在 `references/style-profiles/<name>.md`，包含：

1. **来源信息**: URL、提取时间、公众号名称
2. **全局设计**: 宽度、背景、字体、色调
3. **正文样式**: 完整 CSS
4. **标题系统**: 各级标题 HTML + CSS 模板
5. **高亮样式**: 所有高亮方式的 CSS
6. **图片样式**: 内容图、头图、尾图的 HTML + CSS
7. **特殊组件**: 引用块、列表、卡片等的 HTML + CSS 模板
8. **间距规律**: 各类元素的 margin/padding 总结

## 注意事项

1. **保留原始 HTML 结构**: 公众号编辑器对 HTML 有严格要求，保留源文章经过验证的结构
2. **inline CSS**: 所有样式必须 inline，公众号不支持 `<style>` 标签
3. **图片处理**: 需要单独处理图片上传（base64 或 URL）
4. **font-size 放在 `<p>` 上**: 公众号 ProseMirror 编辑器会剥离 `<span>` 上的 font-size
5. **text-align 放在块级元素上**: inline 元素上的 text-align 无效

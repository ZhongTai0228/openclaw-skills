## 公众号排版 - YC Agent Economy 文章

### 关键发现
- **`text-align: justify` 必须写在 `<p>` 上**，写在 `<span>` 上完全无效（CSS 规范：text-align 只对块级元素生效）
- `word-break: break-all` 太暴力，会从英文单词中间劈开
- `overflow-wrap: break-word` 也不对，不是根本解决方案
- 正确做法：只在 `<p>` 上写 `text-align: justify`，不加任何额外断词属性，浏览器默认行为就是最佳效果

### 图片提取方法
- `fetch()` + `credentials: 'include'` 从飞书页面同源请求 = 稳定可靠
- `new Image()` + `crossOrigin='use-credentials'` + canvas = CORS 失败，废弃
- `feishu_drive_fetch_media` = 缺 app scope，不可用

### 公众号编辑器
- `#title` 是 `<textarea>`，必须用 native setter 设置
- 不设标题会导致 `default-203` 保存错误
- 预览发送不受保存失败影响（独立操作）
